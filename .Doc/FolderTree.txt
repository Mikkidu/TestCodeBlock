СВЯЗИ И ЗАВИСИМОСТИ:

GameManager (Managers)
  │
  ├─→ RobotController (Robot)
  │   └─→ RobotConfig (ScriptableObjects/Configs)
  │
  ├─→ CommandExecutor (Execution)
  │   ├─→ ICommand (Core)
  │   └─→ ExecutionContext (Execution)
  │
  ├─→ BlockPalette (UI)
  │   ├─→ BlockFactory (UI)
  │   └─→ BlockUI (UI)
  │
  └─→ ProgramArea (UI)
      ├─→ ProgramSequence (Execution)
      └─→ BlockUI (UI)

РЕКОМЕНДУЕМЫЙ ПОРЯДОК СОЗДАНИЯ АССЕТОВ:

1️⃣  Создать Robot.prefab
    └─ Cube + DirectionIndicator

2️⃣  Создать RobotConfig.asset
    └─ Настроить параметры

3️⃣  Создать Canvas.prefab
    └─ Canvas + EventSystem

4️⃣  Создать BlockUI.prefab
    └─ Panel + Image + Text + CanvasGroup

5️⃣  Создать Sprites для команд
    └─ 5 иконок разных цветов

6️⃣  Создать BlockPalette.prefab
    └─ ScrollView + Grid Layout

7️⃣  Создать ProgramArea.prefab
    └─ Panel + Vertical Layout

8️⃣  Создать звуки (опционально)
    └─ 5-7 звуковых файлов

9️⃣  Собрать полную сцену в GameScene.unity
    └─ Все компоненты + GameManager

РАЗМЕР И МАСШТАБ:

📊 Примерные объёмы:
  • Всего папок: 15+
  • Всего файлов: 50-80 (в зависимости от контента)
  • Размер проекта: 200-500 MB (с исходниками)

🎯 Для минимального прототипа:
  • Обязательно: Scenes, Prefabs, ScriptableObjects, Scripts
  • Опционально: Sprites, Sounds, Materials
  • Можно добавить позже: Resources, Editor
